    Ubuntu网络设置

    Permalink: 2013-04-15 21:35:00 by hyhx2008in ubuntu tags: ubuntu linux network

    1.配置ip

    ubuntu的网络配置信息放在 /etc/network/interfaces 中;

    如果配置动态获取ip，则在上述文件中加入以下内容：

    auto eth0
iface eth0 inet dhcp

    如果配置静态ip，则添加如下内容：

    auto eth0
iface eth0 inet static
address 192.168.1.155
netmask 255.255.255.0
gateway 192.168.1.1

    要是配置生效，需要重启网卡：

    ifconfig eth0 down
ifconfig eth0 up

    接着用ifconfig命令查看ip是否配置成功，若还有没有配置成功，则需重启下网络服务

    VMware虚拟机启动时总提示cann't connect to floopy0的问题

    Permalink: 2012-11-13 17:26:00 by hyhx2008in tech tags: ubuntu VMware

    在VMware中装了ubuntu，启动时总提示不能连接到设备floopy0，关也关不掉，很讨厌。

    百度了一下解决办法: 修改虚拟机的.vmx文件。

    1.将floppy0.autodetect这项改为FALSE:

    floppy0.autodetect = "FALSE"

    2.添加一句:

    floppy0.startConnected = "FALSE"

    The End!!

    重新安装ubuntu后的todo list

    Permalink: 2012-11-11 21:44:00 by hyhx2008in ubuntu tags: ubuntu

    经常把虚拟机里ubuntu搞坏，重装后总记不清还要做什么设置，安什么软件。

    写在这里做个提醒吧，以后再慢慢添加。

    1.安装aptitude

    $sudo apt-get install aptitude

    2.安装vim

    $sudo aptitude install vim

    并修改/usr/share/vim/vimrc 。

    3.添加sudoers

    $sudo visudo

    加入 hy ALL = （ALL）ALL。

    ubuntu下好像这样做后还是不能使用su命令切换到root，百度解决办法:

    $sudo passwd

    修改一下密码就好了。

    4.安装openssh

    为了能让secureCRT连上虚拟机里的ubuntu，需要安装openssh。

    $sudo aptitude install openssh-server openssh-client

    再用ifconfig查一下ip，填到secureCRT里就好了。
