    让google和百度检索自己的网站

    Permalink: 2012-11-18 23:58:00 by hyhx2008 in tech tags: google baidu

    自从建立起github博客也有两个星期了，但是在google上一直搜索不到自己的博客。

    等了这么久google还没有抓取我的博客内容。今天问了马星这个问题，马星又早就把解决办法贴在他的 blog 上了。。

    其实可以主动向搜索引擎申请检索自己的网站，申请链接如下:

    google: http://www.google.com/addurl/?hl=zh-CN&continue=/addurl

    baidu: http://www.baidu.com/search/url_submit.html

    填写自己的网址即可。

    (问完马星才发现google好像几天前已经检索了我的博客内容。。。)

    马星那那篇 blog 上还讲了如何添加google自定义搜索，不过要使用bootsrap2主题。。

    我瞎折腾了一会儿，也在左边添加了个畸形的自定义搜索框。。。

    The End！！

    VMware虚拟机启动时总提示cann't connect to floopy0的问题

    Permalink: 2012-11-13 17:26:00 by hyhx2008 in tech tags: ubuntu VMware

    在VMware中装了ubuntu，启动时总提示不能连接到设备floopy0，关也关不掉，很讨厌。

    百度了一下解决办法: 修改虚拟机的.vmx文件。

    1.将floppy0.autodetect这项改为FALSE:

    floppy0.autodetect = "FALSE"

    2.添加一句:

    floppy0.startConnected = "FALSE"

    The End!!

    重新安装ubuntu后的todo list

    Permalink: 2012-11-11 21:44:00 by hyhx2008 in tech tags: ubuntu

    经常把虚拟机里ubuntu搞坏，重装后总记不清还要做什么设置，安什么软件。

    写在这里做个提醒吧，以后再慢慢添加。

    1.安装aptitude

    $sudo apt-get install aptitude

    2.安装vim

    $sudo aptitude install vim

    并修改/usr/share/vim/vimrc 。

    3.添加sudoers

    $sudo visudo

    加入 hy ALL = （ALL）ALL。

    ubuntu下好像这样做后还是不能使用su命令切换到root，百度解决办法:

    $sudo passwd

    修改一下密码就好了。

    4.安装openssh

    为了能让secureCRT连上虚拟机里的ubuntu，需要安装openssh。

    $sudo aptitude install openssh-server openssh-client

    再用ifconfig查一下ip，填到secureCRT里就好了。
